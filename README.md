# Tutoriais de Servidor Caseiro com Termux

Coleção de tutoriais em português para transformar celulares Android em servidores usando o Termux.

## Artigos

| Tutorial | Descrição |
|---|---|
| [Servidor NAS Caseiro](tutorial_servidor_nas.md) | Transforme um celular antigo em um servidor NAS para armazenar e acessar arquivos na rede local |
| [Servidor de Músicas](tutorial_musicas_online.md) | Configure o Navidrome no Termux para streamer sua biblioteca musical |
| [Servidor de Minecraft](tutorial_servidor_minecraft.md) | Rode um servidor de Minecraft (Paper MC 1.20.4) diretamente no celular |

## Pré-requisitos gerais

- Android com [F-Droid](https://f-droid.org/) instalado
- [Termux](https://f-droid.org/packages/com.termux/) instalado via F-Droid
- Termux atualizado (`pkg update && pkg upgrade -y`)

## Objetivo

Dar uma segunda vida a celulares antigos, evitando o descarte eletrônico e aproveitando o hardware para rodar serviços úteis no dia a dia.
