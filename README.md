## Pré-requisitos gerais

- Android com [F-Droid](https://f-droid.org/) instalado
- [Termux](https://f-droid.org/packages/com.termux/) instalado via F-Droid
- Termux atualizado (`pkg update && pkg upgrade -y`)

## Objetivo

Dar uma segunda vida a celulares antigos, evitando o descarte eletrônico e aproveitando o hardware para rodar serviços úteis no dia a dia.
